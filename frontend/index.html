🚀 How to use with Seestar S50:
1. Click "seestar_alp CSV" to download your observing plan
2. Set up seestar_alp on a Raspberry Pi or computer
3. Connect your Seestar S50 to WiFi (Station Mode)
4. Import the CSV file in seestar_alp web interface
5. Start automated imaging - it's that simple!
The official Seestar app has no import capability. seestar_alp is the community standard for automated observing.

📱 Share Observing Plan

How to use:

  1. Scan the QR code with your phone or tablet
  2. Opens a shareable web page with your plan
  3. Download CSV for seestar_alp import
  4. Share the URL with others
💡 The QR code contains a short URL to your plan. Plans are stored temporarily on this server.
